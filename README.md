# ✂ Cut the Fat

A personal finance web app that turns bank statement uploads into actionable cost-cutting advice. Not just "here's where your money went" — **here's what you should stop paying for.**

---

## Features

- **Drag-and-drop upload** — CSV, Excel (.xlsx/.xls), and PDF bank statements
- **AI categorization** — Claude Haiku auto-assigns every transaction to a category; merchant rules persist so known merchants are never sent to the API again
- **Inline correction** — click any category to reassign it; the rule is saved for all future uploads
- **Dashboard** — monthly spend by category with vs-last-month comparison
- **History** — trend charts across the last 3–12 months
- **Transactions** — searchable, filterable list with pagination
- **Insights** — 5 plain-language cost-cutting tips generated by Claude Sonnet, cached and invalidated automatically when data changes
- **Single-user auth** — password-protected with a persistent session token

## Quick Start

### 1. Prerequisites

- Python 3.11+
- Node.js 18+
- An [Anthropic API key](https://console.anthropic.com/)

### 2. Configure environment

```bash
cp .env.example .env
```

Edit `.env`:

```env
ANTHROPIC_API_KEY=sk-ant-...
APP_PASSWORD=your-secure-password
SECRET_KEY=a-random-string-at-least-32-chars
```

### 3. Run

```bash
./start.sh
```

`start.sh` will:
1. Create a Python virtual environment (first run only)
2. Install backend dependencies
3. Run any pending database migrations
4. Start the FastAPI backend on `http://localhost:8000`
5. Install frontend dependencies (first run only)
6. Start the Vite dev server on `http://localhost:5173`

Open **http://localhost:5173** and sign in with the password from your `.env`.

> API docs (Swagger UI): http://localhost:8000/docs

## How It Works

### Upload pipeline

```
Browser drops file
  → POST /api/uploads (multipart)
  → SHA-256 duplicate check
  → Parser dispatch (CSV / Excel / PDF)
  → Normalize merchants, compute dedup hashes, skip existing
  → Apply saved merchant rules  (no API call)
  → Batch-call Claude Haiku for unknown merchants (~1 call per upload)
  → Bulk insert transactions
  → TanStack Query invalidates all cached data
  → Dashboard refetches → charts update
```

### Insights cache

Insights are keyed on `SHA-256(aggregated_spend_json)`. When you upload a new statement, the data changes, the hash changes, and the next `/api/insights` call regenerates fresh tips from Claude Sonnet. Hit the **Refresh Insights** button or pass `?force=true` to bypass the cache manually.

### Merchant rules

Every time you correct a category inline, the `merchant_normalized → category` mapping is stored in `merchant_rules`. The next upload checks this table before calling Claude, so frequently-seen merchants never cost API tokens.

## API Reference

| Method | Path | Description |
|---|---|---|
| `POST` | `/api/auth/login` | `{password}` → `{token}` |
| `GET` | `/api/auth/verify` | Check token validity |
| `POST` | `/api/uploads` | Upload + parse + categorize statement |
| `GET` | `/api/uploads` | List past uploads |
| `DELETE` | `/api/uploads/{id}` | Remove upload and its transactions |
| `GET` | `/api/transactions` | List with filters: `month`, `category`, `search`, `type` |
| `PATCH` | `/api/transactions/{id}/category` | Reassign category + save merchant rule |
| `GET` | `/api/transactions/export` | CSV download |
| `GET` | `/api/dashboard/summary` | Category totals for a month |
| `GET` | `/api/dashboard/comparison` | Current vs previous month delta |
| `GET` | `/api/dashboard/history` | Monthly totals per category (last N months) |
| `GET` | `/api/insights` | AI cost-cutting tips (cached) |

## Environment Variables

| Variable | Default | Description |
|---|---|---|
| `ANTHROPIC_API_KEY` | *(empty)* | Required for AI features. Without it, categorization defaults to "Other" and insights use rule-based fallbacks. |
| `APP_PASSWORD` | `changeme` | Login password |
| `SECRET_KEY` | `change-this-secret-key-in-production` | HMAC signing key for auth tokens |
| `DATABASE_URL` | `sqlite+aiosqlite:///./cut_the_fat.db` | SQLAlchemy async DB URL |

## Development

### Backend only

```bash
cd backend
source .venv/bin/activate
uvicorn app.main:app --reload --port 8000
```

### Frontend only

```bash
cd frontend
npm run dev
```

### Run a migration

```bash
cd backend
.venv/bin/alembic revision --autogenerate -m "description"
.venv/bin/alembic upgrade head
```

### Type-check frontend

```bash
cd frontend
npx tsc --noEmit
```
