Ausgabenübersicht anzeigen mit dem Cut the Fat CLI.

Führe aus:
```bash
cd /Users/ecog-vladislav/Projects/cut-the-fat && ./ctf dashboard $ARGUMENTS
```

Zeige die Ausgabe. Ohne Argument wird der letzte Monat mit vorhandenen Daten angezeigt. Mit `--monat JJJJ-MM` kann ein bestimmter Monat gewählt werden.
