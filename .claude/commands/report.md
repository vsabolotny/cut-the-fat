Monatsbericht als Markdown-Datei generieren.

Führe aus:
```bash
cd /Users/ecog-vladislav/Projects/cut-the-fat && ./ctf report $ARGUMENTS
```

Zeige die Ausgabe. Dann lese die generierte Datei `analytics/JJJJ-MM.md` und zeige deren Inhalt an.

Optionen:
- `--monat JJJJ-MM` für einen bestimmten Monat
- `--alle` für alle verfügbaren Monate
